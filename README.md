
Que es la practica y que aprendimos
La practica crea un detector de spam que revisa mensajes y dice si son seguros o no
Aprendimos a usar python git github y a trabajar con archivos modulos y librerias

Como lo hicimos y tecnologias usadas
Creamos el entorno virtual instalamos librerias con pip y organizamos el codigo en carpetas
Usamos pandas y scikit learn para procesar datos y entrenar el modelo
Subimos el proyecto a github con comandos basicos de git

Trabajo futuro y mejoras
Podemos mejorar el modelo usar mas datos y crear una interfaz mejor
Se pueden corregir errores reinstalando librerias o revisando los modulos
Tambien se puede convertir el programa en app web o movil.
